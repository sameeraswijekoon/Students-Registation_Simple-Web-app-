# 🎓 Educational Platform - Course Management System

An interactive web platform for managing university degree programs, course enrollments, and student registrations across multiple faculties.

<img src="https://github.com/user-attachments/assets/45cf73b0-c937-4a38-8235-2b3dc6b344d8" width="80%" height="80%">

## 📚 Features Overview

### 🏠 Homepage (Index)
- Interactive faculty showcase
- Quick access to all programs
- Modern navigation system
- Student resources section
- Dynamic course highlights

### 🔐 Authentication System

#### Login Features
- Secure user authentication
- Email/Username login support
- "Remember Me" functionality
- Password recovery option
- Error handling and validation
- Redirect to registration

#### Registration Features
- New user account creation
- Email verification
- Password strength checker
- Student ID integration
- Form validation
- Security measures (CAPTCHA)

### 📝 Student Management

#### Course Registration
- Interactive registration form
- Real-time course availability
- Semester selection
- Prerequisites checking
- Fee structure display
- Application tracking

#### Course Catalog
- Faculty-wise course listing
- Detailed program information
- Interactive course details
- Application deadlines
- Fee structures
- Prerequisites display

## 🔧 Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/education-platform.git
```

2. Navigate to project directory
```bash
cd education-platform
```

3. Open index.html in your browser or set up a local server

## 💻 Usage

### For Students
1. Create new account via Sign-up
2. Login with credentials
3. Browse available courses
4. Register for desired programs
5. Track applications

### For Administrators
1. Access admin panel
2. Manage course listings
3. Process registrations
4. Update course information
5. Monitor enrollments

## 📊 Database Structure

```
users/
├── students/
│   ├── personal_info
│   ├── enrolled_courses
│   └── application_status
│
├── courses/
│   ├── course_details
│   ├── prerequisites
│   └── enrollment_count
│
└── faculties/
    ├── programs
    ├── course_listings
    └── faculty_info
```
